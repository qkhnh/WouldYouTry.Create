# PrepBrain - Marketing Overview

## Product Summary

**PrepBrain** is an AI-powered kitchen assistant designed for cafes, brunch spots, and small restaurants. It helps chefs create daily specials from leftover ingredients, reducing food waste while maximizing profitability.

---

## The Problem We Solve

- **Food waste costs money** — Fresh ingredients that don't get used end up in the bin
- **Daily specials are time-consuming** — Chefs spend valuable time brainstorming what to cook
- **Inconsistent menu planning** — Without a system, at-risk ingredients get overlooked
- **No easy way to track favorites** — Successful dishes get forgotten instead of reused

---

## Key Features

### 🎤 Voice-Powered Ingredient Entry
Hands-free input during busy morning prep. Simply speak your ingredients and quantities — "500 grams of tomatoes, 3 avocados, 200 grams of bacon" — and PrepBrain automatically selects them.

### 🤖 AI-Generated Dish Suggestions
Powered by GLM-4 AI, PrepBrain generates 3 creative off-menu dish ideas tailored to:
- Your cafe's cuisine style
- Available kitchen equipment
- Existing menu items
- Ingredients at risk of spoiling

### ⚠️ At-Risk Ingredient Prioritization
Mark ingredients that need to be used today. PrepBrain prioritizes these in its suggestions and calculates a **Waste Score** showing how effectively each dish uses at-risk items.

### 📊 Smart Portion Calculator
Automatically estimates how many portions can be made from available ingredients, helping you plan for the day.

### 💾 Recipe Library
- **Save** dishes you love for future use
- **Approve** dishes for today's special
- **View** detailed recipes with ingredient lists

### 📱 Menu Sharing
Share your approved specials with customers via a generated menu link.

### 🏪 Cafe Profile Customization
Set up once and get tailored suggestions:
- Cafe name and cuisine type
- Available equipment (stovetop, oven, grill, fryer, blender, etc.)
- Existing menu items for style reference
- Pantry staples always available
- Ingredient preferences and restrictions

## User Journey

1. **Sign Up** → Create account via email
2. **Onboard** → Set up cafe profile (equipment, style, menu)
3. **Daily Prep** → Enter leftover ingredients (voice or manual)
4. **Get Ideas** → Receive 3 AI-generated dish suggestions
5. **Approve** → Choose today's special
6. **Share** → Publish to customers
7. **Save** → Keep winners for future use
