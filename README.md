#  WouldYouTry.create

An AI-powered tool for brunch cafés to turn leftover ingredients into creative, off-menu dish ideas.


##  Overview

**WouldYouTry.create** is the management-side application of our hackathon project.  
It helps café owners reduce food waste and experiment with new dishes by generating recipe ideas based on available leftover ingredients.

Instead of discarding unused ingredients or sticking to repetitive menus, owners can quickly create new, realistic dishes aligned with their café’s style.


##  Problem

Cafés face daily challenges:

- Leftover ingredients often go to waste  
- Menu innovation is risky and time-consuming  
- Staff lack time to experiment with new recipes  
- Creativity is limited by operational pressure  


##  Solution

WouldYouTry.create allows café owners to:

- Input available leftover ingredients  
- Generate AI-powered dish suggestions  
- Get structured recipes instantly  
- Experiment with low risk  


##  Features

###  Ingredient Input System

- Select ingredients from a predefined list  
- Adjust quantities and units  
- Add notes (e.g. "use today", "expiring soon")  


###  AI Dish Generation

- Generates dish ideas based on:
  - Selected ingredients  
  - Café style (captured during onboarding)  

- Produces:
  - Dish name  
  - Description  
  - Ingredients used  
  - Cooking instructions  
  - Portion scaling  

###  Context-Aware Suggestions

- Matches suggestions to menu style  
- Avoids unrealistic combinations  
- Focuses on practical, cookable dishes  


###  Save & Approve Recipes

- Owners can:
  - Save generated dishes  
  - Approve dishes for use  

- Approved dishes can be pushed to the social platform (**WouldYouTry**)
